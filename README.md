## 📖 About MeowHug
**MeowHug** is a modern, high-end e-commerce platform dedicated exclusively to feline fashion. Designed with a premium pastel green and white aesthetic, the website features cinematic layouts, interactive scroll-driven animations, and a seamless user experience. Whether you're looking for tiny sweaters or cute collars, MeowHug provides a "purr-fect" shopping journey for every cat lover! 🐾✨

## ✨ Core Features
- 🎥 **Cinematic Hero Section:** Immersive looping video background with custom fade-in/fade-out logic.
- 🖼️ **Interactive Video Scrolling:** Advanced horizontal parallax effects for a high-end product display.
- 🎭 **Curtain Reveal Footer:** Sticky, fixed-position footer that reveals beautifully as you scroll to the bottom.
- 🛍️ **Dynamic Cart & Filtering:** Easily filter products by cat size and manage your cart seamlessly.
- 🌗 **Theme Toggles:** Built-in Dark/Light mode integration for a comfortable viewing experience.
- 📱 **Fully Responsive:** Optimized for both desktop and mobile devices with modern typography (Instrument Serif & Inter).

## 🛠️ Tech Stack
- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Custom Pastel Green/White Theme)
- **Animations:** Custom `requestAnimationFrame` & CSS Transitions
